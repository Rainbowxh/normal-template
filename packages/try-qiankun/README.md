1. 加载不同的子应用到指定的位置 (子应用位置)

1. 隔离各个子应用的环境变量、 

----

## 子应用
子应用的代码会作为html渲染到目标位置

```html
<html>
  <body>
    ....
    <div id="app">
      <html></html>
    </div>
  </body>
</html>
```

1. 导出三个必要的声明周期
bootstrap: 渲染之前进行部分操作

mount:     渲染函数

unmount:   卸载函数
